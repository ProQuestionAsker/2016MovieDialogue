Gender Equality and Dialogue in 2016’s Highest Grossing Films

In 2016, several new releases contained a female lead, but did females occupy as many roles or speak as much as male characters? To find out, I analyzed the word count for every speaking character in 2016’s Top 10 Worldwide Highest Grossing Films.

Every circle in the figure to the right represents a single speaking character. Each circle is scaled based on the number of words spoken by that character.
